
Este proyecto es una página web donde puedes buscar personajes de la serie Rick y Morty usando una API pública. Permite filtrar por nombre,  cambiar entre modo claro/oscuro y navegar con paginación.

Funciones principales

Buscar personajes por nombre

Paginación (anterior y siguiente)

Modo claro/oscuro guardado en localStorage

Tarjetas con imagen, especie, género, estado y origen

Totalmente responsivo (celular, tablet y PC)

Tecnologías usadas

HTML5

CSS3 (variables, grid, responsive)

JavaScript Vanilla (fetch, DOM, eventos)

API Rick & Morty: https://rickandmortyapi.com/

Estructura del proyecto
index.html
style.css
index.js
README.txt

Cómo usar la página

Escribe un nombre en el buscador
Haz clic en "Buscar"

Usa la paginación para cambiar de página

Activa el modo oscuro desde el interruptor
